# Study Room Booker

## Description

The Study Room Booker application returns all the available rooms that can be booked on the current day in the Alan Gilbert Learning Commons.

## Installation

1. Install the required dependencies:
    ```sh
    pip install -r requirements.txt
    ```

2. Run the application:
    ```sh
    ./run.sh
    ```

## Usage

When you run the application, you will be prompted to input a username and password. Follow the on-screen instructions to proceed.
